MELODY 튜토리얼

멜로디를 똑똑하게 사용하는 방법

명령어를 통해 봇과 상호작용

봇을 서버에 추가 시킨 후에 "!!tutorial" 명령어를 쳐보세요
간략하게 명령어를 볼 수 있습니다.

명령어 목록

  • !!tutorial : 멜로디를 위한 간단한 튜토리얼
  • !!tutorial ko : 멜로디를 위한 간단한 튜토리얼(한글)
  • !!summon : 보이스 채널로 멜로디 소환하기
  • !!play 'youtubelink' : 유튜브 주소 오디오 재생하기

플레이리스트 만들기

  • !!playlist create '플레이리스트_이름' '비밀번호' : 플레이리스트 만들기

    • ex) !!playlist create madmovie_bgm 1234

  • !!playlist add '플레이리스트_이름' '비밀번호' '유튜브_주소' : 플레이 리스트에 노래 추가하기

    • ex) !!playlist add madmovie_bgm 1234 https://www.youtube.com/watch?v=1Vhf_ZpXJY8

  • !!playlist show '플레이리스트_이름' : 플레이리스트 확인하기

    • ex) !!playlist show madmovie_bgm

  • !!!playlist load '플레이리스트_이름' : 플레이리스트 불러오기

    • ex) !!playlist load madmovie_bgm
