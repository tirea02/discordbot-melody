Melody Tutorial

How to use Melody bot wise

interact with bot by command line

After adding bot to your server please try "!!tutorial" command it will briefly show command list.

list of command

  • !!tutorial : simple tutorial for Melody
  • !!tutorial ko : simple tutorial for Melody in Korean
  • !!summon : summon melody to my voice channel
  • !!play 'youtubelink' : play youtube link audio

playlist create

  • !!playlist create 'playlist_name' 'password' : create your play list

    • ex) !!playlist create madmovie_bgm 1234

  • !!playlist add 'playlist_name' 'password' 'youtube_link' : add play list

    • ex) !!playlist add madmovie_bgm 1234 https://www.youtube.com/watch?v=1Vhf_ZpXJY8

  • !!playlist show 'playlist_name' : check your playlist

    • ex) !!playlist show madmovie_bgm

  • !!!playlist load 'playlist_name' : load playlist

    • ex) !!playlist load madmovie_bgm
