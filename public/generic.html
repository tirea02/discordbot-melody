Melody Tutorial

How to use Melody bot wise

interact with bot by command line

After adding bot to your server please try "!!tutorial" command it will briefly show command list.

list of basic command

  • !!language : select server language 1.Korean 2.English
  • !!tutorial : simple tutorial for Melody
  • !!help : DM to user Melody tutorial
  • !!summon : summon melody to my voice channel
  • !!patchnote : show lastest patch note
  • !!my : show all my playlist

List of play command

  • !!play 'youtubelink' : play youtube link

    • Why there is 2 case?

    There is two type of youtube link
    1.sigle video link
    2.playlist link.
    3. text to search in youtube

    Case1 : single video link -> play single song

    ex) !!play https://www.youtube.com/watch?v=MhQKe-aERsU

    Case2 : youtube playlist of video -> play all song in playlist

    ex) !!play https://www.youtube.com/watch?v=MhQKe-aERsU&list=RDMhQKe-aERsU

    Case3 : text to search on youtube -> search text in youtube and play top video

    ex) !!play tangled i see the light

  • !!now : show enqueued songs "
  • !!pause : Melody pauses playing song
  • !!resume : Melody resume paused song
  • !!skip : Melody skip now playing song and play next enqueued song
  • !!stop : stop melody's action and kick out melody from voice channel

List of playlist command

  • !!playlist create 'playlist_name' : create your play list

    • ex) !!playlist create disney

  • !!playlist add 'playlist_name' 'youtube_link' : add play list

    • ex) !!playlist add disney https://www.youtube.com/watch?v=J0lG8YorBsU

  • !!playlist show 'disney' : check your playlist

    • ex) !!playlist show madmovie_bgm

  • !!!playlist load 'playlist_name' : load playlist

    • ex) !!playlist load disney

  • !!playlist search 'playlist_name' : List all playlist with their creater. You can select and load it by typing index of playlist.

    • ex) !!playlist search disney

  • !!playlist delete 'playlist_name' 'index number' : Delete song with index number

    • ex) !!playlist delete disney 3 -> delete number 3 song in disney playlist
